## [0.1.0] - 2026-05-26

### Added

- First tagged release. Static analyzer for agent-SDK repositories (Claude
  Agent SDK, OpenAI Agents SDK, Google ADK, MCP) with human / JSON / SARIF
  output.
- Release distribution via GitHub Releases, Homebrew tap, Scoop bucket, and a
  multi-arch `ghcr.io/trustabl/trustabl` Docker image.
